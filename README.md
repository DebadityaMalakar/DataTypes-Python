## Int
This datatype is the extension of the original integer datatype in python and has different integer datatypes from the primitive times.
- Int2(2 Bit Integer)
- Int4(4 Bit Integer)
- Int8(8 Bit Integer)
- Int16(16 Bit Integer)
- Int32(32 Bit Integer)
- Int64(64 Bit Integer)
